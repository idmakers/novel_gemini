# 《雪與鋼琴的邊境》小說設計規格說明書 (Boy Meets Girl)

本設計文檔確立了一部融合《月姬裏篇》、《86-不存在的戰區》、《進擊的巨人》、《白色相簿2》與《物語系列》價值觀及設定要素的十萬字「純愛 Boy Meets Girl」小說的設計細節。

---

## 1. 核心設定與世界觀

### 1.1 世界構造：高牆與冰原
* **箱庭 (The Box Garden)**：
  * 人類最後的庇護所，由三重高聳的巨牆（外圈「白銀」、中圈「黃金」、內圈「琉璃」）圍繞。
  * 牆內四季如春，文化藝術高度繁衍。上層貴族崇尚古典鋼琴與歌劇。
  * **黑幕**：高牆並非天然或神蹟，其結界防護力來自「共鳴儀（Symphony）」的逆向能量抽取——即提取在牆外戰死的「被同化者」靈魂與執念作為燃料。
* **永凍荒原 (The Frozen Wasteland)**：
  * 牆外的世界。氣溫長年低於零下三十度，暴風雪永無止息。
  * 傳說雪原的盡頭是「海」——沒有冰凍、波濤起伏的無邊水域，象徵著虛無又自由的彼方。

### 1.2 威脅與戰鬥：怪異與同化
* **怪異巨獸 (The Oddities)**：
  * 牆外的怪物。牠們並非普通生物，而是人類集體無意識中的陰暗、罪孽、創傷與悔恨在極寒中的結晶（《物語》元素）。
  * 牠們體型巨大，具有強烈的物理破壞力（《巨人》元素），並且會發出干擾精神的「雜音」。
* **半生物機關「影縫 (Shadow-stitcher)」**：
  * 前線死士駕駛的戰鬥機甲。其核心是一顆人工封印的「怪異心臟」。
  * 戰鬥時，駕駛員必須以自身的精神、記憶和情感為代價（情感獻祭），與機關達成同調。
  * **同化詛咒 (Assimilation Curse)**：隨著戰鬥次數增加，駕駛員會產生「反轉衝動」（《月姬》元素），身體開始金屬化或結晶化，且會逐步喪失人類感官（忘記甜味、色彩、樂音，最後忘記人的名字），最終徹底化為失去理性的怪異巨獸。

### 1.3 精神連結：共鳴儀 (Symphony)
* 牆內的「守望者（Handler）」與牆外的「駕駛員」之間的精神對調系統（《86》元素）。
* 守望者可以共享駕駛員的視聽，並通過傳輸音樂或語言，充當駕駛員精神的「錨」，減緩其同化進度。
* **精神污染**：守望者在連結時會承受駕駛員大腦中累積的殺戮衝動與痛覺，時間過長會導致精神崩潰。

## 2. 角色設定

### 2.1 男主角：修亞 (Shua)
* **身份**：前線防衛隊第一小隊王牌，代號「無名骨 (Lipless)」。
* **性格**：說話極具《物語》系列主角的自嘲、口是心非與病態的敏銳，內心冷靜卻對自由有著《86》與《巨人》般近乎偏執的追求。
* **同化進度**：感官功能已喪失30%。開始無法嗅到香味，聽覺的高頻區段被雜音覆蓋。
* **機甲**：駕駛「影縫」，擅長近戰格鬥。

### 2.2 女主角A（牆內）：塞西莉雅 (Cecilia)
* **身份**：帝國大貴族千金，天賦異稟的帝國鋼琴家，代號「白銀之琴」。
* **性格**：優雅、高潔，不諳世事，卻有著堅韌的內在。
* **定位**：修亞的精神守望者。在彈奏鋼琴守護修亞的過程中被他的靈魂吸引。隨著對高牆黑幕的揭露，她陷入了自我存在價值崩塌的罪惡感與痛苦中（《白相2》式的精神糾葛）。

### 2.3 女主角B（牆外）：蕾拉 (Layla)
* **身份**：修亞的僚機駕駛員，代號「鏽刃 (Rusty Blade)」。
* **性格**：幹練、毒舌，與修亞互相以冷嘲熱諷來掩蓋對死亡的恐懼。
* **定位與詛咒**：右臂已完全金屬化。與修亞共享戰場上的血腥與寒冷。她深愛修亞，理解他的靈魂被牆內的鋼琴聲所吸引，對此感到嫉妒，但又感激塞西莉雅能讓修亞維持人形。她代表著與男主「共死」的物理現實。

## 4. 寫作核心原則

1. **對白風格**：修亞與蕾拉的對話需展現西尾維新式的字詞聯想、雙關、自我否定與快速節奏，用以掩飾末世的創傷。
2. **胃痛感（白色相簿2）**：修亞對塞西莉雅是「靈魂上的救贖與可望不可及的白月光」，對蕾拉是「肉體與命運上的交纏與無可逃避的共死」。三人的感情交織著隱瞞、愧疚、嫉妒與深沉的純愛。
3. **殘酷度（86/巨人）**：高牆內外的階級壓迫、死士被當作「消耗性零件」的非人待遇、以及追求自由時必須付出的血腥代價。
4. **意象使用**：鋼琴、暴雪、鐵鏽、金屬化的皮膚、乾枯的眼淚、沒有封凍的「海」。
