# 第一章小說滲透式擴寫 實現計劃

> **面向 AI 代理的工作者：** 必需子技能：使用 superpowers:subagent-driven-development（推薦）或 superpowers:executing-plans 逐任務實現此計劃。步驟使用複選框（`- [ ]`）語法來跟蹤進度。

**目標：** 擴寫小說第一章 `chapters/chapter_01.md`，新增 2000~2500 字細節、心理及對白，使總字數達到 10000 字以上（不超過 12000 字），且通過 `verify_chapter.py` 驗證。

**架構：** 採用全章節滲透式擴寫方案（方案 A），對 6 個小節的特定位置進行潤飾與擴增。融入西尾維新風格的哲學碎嘴、反轉衝動的意識流拉扯，以及對塞西莉雅複雜心態的挖掘。

**技術棧：** Markdown, Python 3, Git, verify_chapter.py 驗證腳本

## 檔案與職責

- `chapters/chapter_01.md`：小說第一章主檔案。
- `scripts/verify_chapter.py`：字數與關鍵字驗證腳本。
